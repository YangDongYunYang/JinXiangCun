# 第一章 进销存管理系统

## 项目介绍与亮点

 这是一个基于**SpringBoot 3 + Vue 3**技术栈开发的现代化进销存管理系统，专为中小型企业的供应链管理需求设计。系统采用前后端分离架构，实现了采购、销售、库存管理的全流程数字化，通过角色权限控制为不同职能员工提供定制化工作界面。

# 1.管理端模块

登录：账号密码登录，支持密码重置。

用户管理：查看和管理所有用户信息，支持封禁/解封用户。

角色管理：根据角色分配用户权限。

供应商管理：维护供应商信息、合作状态管理（启用/禁用）支持通过excel批量导入/导出商品数据。

商品管理：商品信息录入（名称、分类、规格、成本价、销售价等）。

商品分类管理：对商品的分类进行管理。

商品库存预警：对低于库存阈值的商品进行提前预警。

采购管理：采购订单创建、审核、修改、取消。

销售管理：销售订单管理。

# 2.员工端模块

登录：用户登录，修改密码。

个人信息：查看和修改个人资料（头像、昵称、简介等）。

业务功能：

1.创建采购订单。

2.查看采购历史记录。

3.供应商查询与联系。

4.创建销售订单。
