<h1>项目介绍</h1>
这是一个基于SpringBoot 3 + Vue 3技术栈开发的现代化进销存管理系统
<h1>技术栈</h1>
后端：SpringBoot3 + JWT + Druid + MyBatis

前端：Vue3 + ElementPlus + Axios

数据库： MySQL

<h1>版本</h1> 

jdk: 17

mysql: 8

node.js:18

maven: 3.9
<h1>管理端模块</h1>
登录：账号密码登录，支持密码重置。

注册：输入账号，密码，确认密码完成注册。

用户管理：查看和管理所有用户信息，支持封禁/解封用户。

角色管理：根据角色分配用户权限。

供应商管理：维护供应商信息、合作状态管理（启用/禁用）支持通过excel批量导入/导出商品数据。

商品管理：商品信息录入（名称、分类、规格、成本价、销售价等）。

商品分类管理：对商品的分类进行管理。

商品库存预警：对低于库存阈值的商品进行提前预警。

采购管理：采购订单创建、审核、修改、取消。

销售管理：销售订单管理。
<h1>员工端模块</h1>
登录：用户登录，修改密码。

个人信息：查看和修改个人资料（头像、昵称、简介等）。

业务功能：

1.创建采购订单。

2.查看采购历史记录。

3.供应商查询与联系。

4.创建销售订单。
